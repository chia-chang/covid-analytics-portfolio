## covid_integrated_data.csv

This file contains the cleaned and fully integrated COVID-19 dataset generated by the Data_Cleaning_and_Integration.ipynb notebook. The dataset aggregates daily COVID-19 statistics from five separate sources, organized into a single, consistent timeline.

### Dataset Overview

- **Number of rows:** 1,432 (one row per day)
- **Number of columns:** 26
- **Date range:** 2020-03-01 to 2024-01-31
- **File size:** ~386 KB

### Key Columns

- `date`: Calendar date for the entry
- `cases`, `tests`, `hospitalizations`, `deaths`, `vaccinations`: Daily counts
- Columns ending `_7day_avg`: 7-day rolling average values
- `positivity_rate`: Daily percent of positive cases among tests
- Week-over-week changes for each metric
- Cumulative totals
